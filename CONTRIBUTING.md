# Contributing to HAVEN Protocol

Thank you for your interest in contributing to the HAVEN Protocol.

## Ways to Contribute

### 1. Protocol Feedback

Open an issue to discuss:
- Clarifications needed in the specification
- Edge cases not addressed
- Potential improvements to the protocol design

### 2. Documentation

- Fix typos or improve clarity
- Add examples
- Translate to other languages

### 3. Implementation

If you're building a HAVEN-compliant implementation, we'd love to hear about it. Open an issue to share your experience.

## Process

1. **Open an Issue First** - For significant changes, open an issue to discuss before submitting a PR
2. **Fork & Branch** - Fork the repo and create a feature branch
3. **Submit PR** - Submit a pull request with a clear description
